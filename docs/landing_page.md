# 합주실 예약 관리 시스템 (Landing Page Content)

## 1. Hero Section
**제목:** 합주실 예약, 아직도 선착순에 맡기고 있나요?
**부제목:** 공정성 · 효율성 · 운영 자동화를 동시에 해결하는 예약 시스템
**액션:** 문의하고 시작하기 (Contact 섹션으로 이동)

---

## 2. 문제 제기 (Problem)
### 이런 문제가 반복되고 있지 않나요?
*   **특정 사용자 독점 예약** → 인기 시간대가 항상 같은 사람에게 돌아감
*   **노쇼 문제** → 예약 후 사용하지 않아 공간이 낭비됨
*   **운영 부담 증가** → 관리자 수동 조정, 공지, 갈등 발생

---

## 3. 해결책 (Solution)
### 우리는 이렇게 해결합니다
*   **예약 횟수 제한 + 대기열 시스템** → 공정한 자원 분배
*   **자동 취소 및 패널티** → 노쇼 최소화
*   **운영 정책 커스터마이징** → 관리 부담 감소

---

## 4. 활용 사례 (Use Case)
### 이런 환경에서 특히 효과적입니다
*   대학교 밴드 동아리 합주실
*   예약 경쟁이 심한 스터디룸
*   회의실 운영이 필요한 조직

---

## 5. 비교 (Comparison)
| 방식 | 문제점 |
| :--- | :--- |
| 구글 캘린더 | 공정성 없음, 선착순 경쟁 |
| 수기 예약 | 관리 비용 증가, 오류 발생 |
| **우리 서비스** | **정책 기반 자동화 + 공정성 확보** |

---

## 6. 서비스 이용 절차 (Process)
1.  **문의하기** → 현재 운영 방식 공유
2.  **운영 방식 상담** → 문제 분석
3.  **맞춤 시스템 세팅** → 정책 적용
4.  **서비스 사용 시작**

---

## 7. 자주 묻는 질문 (FAQ)
*   **Q. 왜 바로 회원가입이 안되나요?**
    *   각 공간마다 운영 방식이 다르기 때문에 맞춤 설정 후 제공됩니다.
*   **Q. 어떤 공간에서 사용할 수 있나요?**
    *   합주실, 스터디룸, 회의실 등 예약 기반 공간에서 사용 가능합니다.
*   **Q. 무료인가요?**
    *   초기 상담 후 운영 규모에 따라 안내드립니다.
