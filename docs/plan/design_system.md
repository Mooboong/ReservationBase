# 디자인 시스템 가이드 (Black & Purple)

사용자가 제공한 "검정 배경에 보라색 포인트" 컨셉을 바탕으로 정의된 디자인 시스템입니다.

## 1. 디자인 시스템 개요
고대비(High Contrast)의 다크 모드 스타일로, 전문적이고 현대적인 느낌을 주며 보라색 포인트를 통해 핵심 액션(CTA)과 하이라이트를 강조합니다.

## 2. 핵심 디자인 요소

### 컬러 팔레트 (Colors)
- **Primary (Purple):** `#A855F7` (핵심 하이라이트, 버튼, 강조 텍스트)
- **Primary Dark:** `#7E22CE` (버튼 호버 시)
- **Background:** `#000000` (전체 배경)
- **Surface:** `#121212` (카드, 섹션 배경)
- **Text Primary:** `#FFFFFF` (기본 본문 및 제목)
- **Text Secondary:** `#9CA3AF` (부연 설명, 비활성 텍스트)
- **Border:** `#262626` (요소 구분선)

### 타이포그래피 (Typography)
- **Font Family:** `Pretendard`, `-apple-system`, `sans-serif`
- **H1 (Hero):** 3.5rem (56px), ExtraBold, Line-height 1.2
- **H2 (Section):** 2.25rem (36px), Bold, Line-height 1.3
- **Body:** 1rem (16px), Regular, Line-height 1.6

### 아이콘 스타일 (Icons)
- **라이브러리:** [Lucide Icons](https://lucide.dev/) (CDN 사용)
- **스타일:** Outline, 2px stroke width
- **색상:** Primary Purple (`#A855F7`) 또는 White

## 3. UI 컴포넌트 스타일
- **Button:** Round 8px, Background Purple, Text White
- **Card:** Round 12px, Background Surface, Border 1px Solid Border color
- **Input:** Background Black, Border 1px Solid Border color, Focus Border Purple
