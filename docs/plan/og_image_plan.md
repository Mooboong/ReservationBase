# 합주인(Hapju-in) Open Graph 이미지 전략 및 프롬프트

Open Graph(OG) 이미지는 링크가 공유될 때(카톡, 슬랙 등) 가장 먼저 보이는 서비스의 얼굴입니다. '합주인'의 신뢰도와 전문성을 높이기 위한 OG 이미지 전략을 제안합니다.

## 1. OG 이미지 구성 요소 (1200 x 630)

### 시각적 레이아웃
- **왼쪽 영역 (Text & Logo):** 서비스 명칭('합주인')과 핵심 슬로건("공정한 합주실 예약의 시작")을 배치합니다.
- **오른쪽 영역 (Visual Accent):** 아이콘 디자인과 일맥상통하는 보라색 네온 그래픽이나 추상적인 음악 장비/파형 이미지를 배치합니다.
- **배경:** 순수 블랙(`#000000`) 또는 아주 짙은 그라데이션을 사용하여 보라색 포인트가 빛나는 느낌을 줍니다.

### 심리적 효과
- **신뢰감:** 깔끔한 레이아웃과 고대비 디자인은 시스템의 안정성과 공정성을 암시합니다.
- **현대적 감각:** 보라색 네온 조명 효과는 트렌디한 IT 서비스라는 인상을 줍니다.
